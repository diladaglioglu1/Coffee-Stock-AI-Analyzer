## Database Architecture

The database layer was implemented using **SQLModel**, an ORM that combines the capabilities of SQLAlchemy and Pydantic to provide type-safe database models and automatic data validation.

The system uses **three relational tables**:

**Product**

Represents inventory items available in the coffee shop.

Key attributes:

- `id` – primary key

- `name` – product name

- `current_stock` – current inventory level

- `unit` – measurement unit (kg, liter, piece)

This table acts as the central entity of the inventory system.

**Sale**

Stores historical sales records used for demand analysis.

Attributes:

- `id`

- `product_id` (foreign key → Product)

- `quantity`

- `date`

Relationship:

```bash
Product (1) ---- (N) Sale
```

This structure allows the system to calculate average sales velocity, which is later used in AI-based stock recommendations.

**Waste**

Tracks inventory loss due to expiration or operational waste.

Attributes:

- `id`

- `product_id` (foreign key)

- `quantity`

- `date`

- `reason`

Relationship:

```bash
Product (1) ---- (N) Waste
```

This data helps simulate real-world inefficiencies in inventory management.

## Historical Data Simulation

To support meaningful inventory analysis, I implemented two mock data generation scripts.

**seed_sales.py**

This script generates approximately 30 days of historical sales data.

The simulation logic includes:

- variable demand patterns

- realistic consumption ranges

- product-specific sales behaviors

This allows the backend to compute sales velocity metrics for each product.

**seed_waste.py**

This script generates records representing inventory loss due to expiration or operational mistakes.

Including waste data helps demonstrate the economic impact of overstocking, which is an important factor in inventory decision-making.

The database layer was designed to support the backend API implemented with **FastAPI**.

These metrics are then forwarded to the Gemini AI module, which generates natural language inventory recommendations.
