## System Integration Architecture

The system follows a full-stack architecture where all components interact seamlessly:
Frontend (React) → Backend (FastAPI) → Database (SQLite) → AI Service (Groq)
## Frontend Layer
Handles user interactions and sends requests to backend APIs
## Backend Layer
Processes requests, interacts with the database, and communicates with the AI service
## Database Layer
Stores inventory, sales, and waste data
## AI Layer
Generates intelligent stock recommendations

---

## Integration Workflow

The integration process works as follows:
1.	The frontend sends a request to the backend when the user clicks the "Analyze" button
2.	The backend retrieves product and sales data from the database
3.	The backend sends this data to the AI service
4.	The AI generates a recommendation
5.	The backend returns the result to the frontend
6.	The frontend displays the result in a modal interface
This workflow enables real-time, dynamic interaction between all system components.
