## Frontend Architecture

The frontend follows a component-based architecture in React:

- Dashboard – manages layout and main state flow  
- ProductCard – displays highlighted products  
- SummaryCard – shows key metrics  
- AnalysisModal – displays AI analysis results  
- ProductListItem – handles product selection  

---

## AI Interaction & Demo System

To ensure system stability, I implemented a dual-mode behavior:

• Uses real backend data when available  

• Falls back to demo-based analysis when backend is unavailable  

This allows the system to remain usable and presentation-ready at all times.

---

## System Integration

The frontend communicates with the backend via REST APIs:

GET /api/stock/products  
POST /api/stock/simulate-day  
GET /api/ai/analyze/{product_id}
